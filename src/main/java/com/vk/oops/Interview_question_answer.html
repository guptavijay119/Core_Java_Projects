🔹 1. Checked Exceptions ✅ Definition: Exceptions that are checked at compile-time. 👉 The compiler forces you to handle them using: try-catch OR throws ✅ Examples: IOException SQLException FileNotFoundException 2. Unchecked Exceptions ✅ Definition: Exceptions that occur at runtime and are not checked at compile-time. 👉 Handling is optional ✅ Examples: NullPointerException ArrayIndexOutOfBoundsException ArithmeticException
